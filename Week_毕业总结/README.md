# 毕业总结

从11月24开课以来，一共10周时间，起码完成了最基本的任务，按时提交作业和笔记，让自己对数据结构和算法有了一个基本认识。

# 学习方法

学习数据结构和算法，最重要的就是像职业训练一样，拆分知识点、刻意练习、寻求反馈（主动、被动）。

- 拆分知识点：

遇到不会的，一定不要死磕，直接看解法，学习解题思路，然后自己默写，多查看优质解法。

- 刻意练习：

要想掌握算法，一定要多练习，切勿只做一遍，实践五毒神掌**（先背，再默，隔日做，隔周做，面试前做）**

- 寻求反馈：

在学习之后，平常的工作中就可以直接运用。

# 经验总结

- 添加自己的注释

自己在学习的时候，只有简单的还有一些思路，大部分题目都是按照题解去默写的，默写的时候，先理解作者的代码，加上自己的注释，这样在下次写的时候，先回想解题思路（注释），然后再写代码。

- 学习过程中一定要坚持，即使你的基础一般，但是只要你坚持下来，就已经比很多人强了（每周人数都在逐渐递减）。
